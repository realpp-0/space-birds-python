# 🐦 Space Birds

A physics-based destruction game inspired by Angry Birds, built from scratch in **Python** using **Pygame** and **Pymunk**.

---

## Features

- Physics-based destruction
- Three unique bird classes
- Breakable materials
- TNT with explosion physics
- Enemy variants
- Dynamic camera system
- Launch trajectory preview
- Bird trails
- Material-specific particle effects
- Sound effects
- Score system
- Eight playable levels

---

## Controls

| Key | Action |
|------|--------|
| Mouse | Aim & launch |
| R | Restart level |
| Esc | Exit level |
| Enter | Continue after winning |
